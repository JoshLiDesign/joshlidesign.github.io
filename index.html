Activity 2: JSON Data - HIV Infeaction Rate By Country

In this activity, I learned how to access JSON files about HIV infection rates by country and display that using graphs. While extracting information is unnecessary for my site, the use of javascript in this activity is important. In sections that utilizes flip-cards or short points, maintaining a consistant spacial design is crucial. Since this activity calculates the width of the screen and bases the size of the bar graph on that, I could employ the same approach to distribute information and spaces evenly across the page. For instance, when the user changes the size of the window, instead of moving elements down, their proportional size would decrease but the overall design would remain the same.
